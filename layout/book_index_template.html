Das Git Community Buch

Willkommen beim Git Community Buch. Dieses Buch wurde erstellt unter Mithilfe Dutzender aus der Git Community und ist zum Selbststudium gedacht, damit man Git möglichst schnell und einfach lernen kann.

Wenn Sie veraltete Informationen entdecken, Verbesserungsvorschläge haben, oder gerne mithelfen würden, dieses Buch weiter zu schreiben, bitte lesen Sie zuerst die How to Contribute-Seite, oder schicken Sie eine E-Mail an den Initiator des Projekts.
